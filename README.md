# 🎵 Jianpu Translator

A lightweight and powerful tool for converting MusicXML into **Jianpu (Numbered Musical Notation)**.

---

## ✨ Overview

**Jianpu-Translator** helps musicians, students, and developers bridge the gap between numeric musical notation (1–7 system) and traditional Western music notation.

It is designed to be:
- Simple to use
- Extensible
- Developer-friendly
- Accurate in musical interpretation

---

## 🚀 Features

- 🎼 Convert Standard staff notation → Jianpu notation 
- 🎹 Export to PNG/PDF format
- 🔊 Playback
- 🧠 Supports octaves, rests, and dotted notes
- ✏️ Editable Note
- ⚙️ Custom adjustment for tempo and key signature
